# STACK

## Core Languages & Runtimes
- **Language:** TypeScript 5.8
- **Runtime:** Node.js (v22 target)

## Frameworks & Libraries
- **Web Framework:** Next.js 14 (App Router)
- **UI & Components:** React 18, `lucide-react`
- **Styling:** Tailwind CSS 3.4, Autoprefixer, PostCSS
- **Animation:** Framer Motion 12
- **Utility:** `date-fns` (dates), `uuid` (unique IDs)
- **Email:** `resend` and `nodemailer`

## Cloud & Infrastructure (AWS)
- `@aws-sdk/client-dynamodb` & `@aws-sdk/lib-dynamodb` for database interactions
- `@aws-sdk/client-cognito-identity-provider` & `amazon-cognito-identity-js` for authentication
- `@aws-sdk/client-ses` for simple email sending
- `@aws-sdk/client-s3` for object storage/assets

## Tooling
- **Script Runner:** `tsx` & `ts-node` for database seeding and utilities
- **Package Manager:** `npm` (implied with CommonJS standard tooling, Next 14)
